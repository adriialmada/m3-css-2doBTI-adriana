Microsoft Excel

Excel es un potente programa de hojas de cálculo desarrollado por Microsoft que permite a los usuarios organizar, analizar y visualizar datos. Utilizado en diversos sectores, Microsoft Excel forma parte del paquete Microsoft Office y admite una amplia gama de funciones, como cálculos, automatización y análisis financiero .

Gnu Cash

es un software libre de contabilidad financiera, de doble entrada y código abierto, diseñado para uso personal y de pequeñas empresas. Permite gestionar cuentas bancarias, acciones, ingresos, gastos, transacciones programadas e informes, siendo una alternativa gratuita a programas comerciales como Quicken

Home Bank

HomeBank es un software de contabilidad personal, gratuito y de código abierto (open source), diseñado para gestionar finanzas, presupuestos, cuentas bancarias y flujo de caja de forma sencilla. Permite importar datos de archivos QIF/OFX/CSV, detectar transacciones duplicadas y visualizar informes. Funciona en Windows, Linux y macOS.
